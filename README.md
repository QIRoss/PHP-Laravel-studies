# PHP flags
-c \<path/to/php.ini> -> custom php.ini file to customize web server configs <br>
-d \<setting=value> -> same as -c flag but dinamically through CLI <br>
-h -> many options <br>
-S -> php built-in web server <br>
-t \<root dir> -> to serve static files <br>
